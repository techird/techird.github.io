概要

本规范定义了叠层样式表（CSS）的 2.1 版本。CSS 2.1 是一个允许用户将样式应用到结构化文档（如 HTML 和 XML）的样式定义语言。分离文档的表现和内容，CSS 2.1 简化了Web开发者的工作以及让Web站点更容易维护。

CSS 2.1 是在 CSS 2 的基础上建立的（CSS 2 在 CSS 1 的基础上建立）。它支持开发人员为不同的媒体类型的设备去裁剪其样式表，包括浏览器、听觉设备、打印机、盲文设备、手持设备等。同时，它还支持内容定位、表格布局、国际化功能以及一些和 UI 相关的属性。

CSS 2.1 修正了 CSS2 中的一些错误（最重要的是重新定义了绝对定位元素的 width 和 height、增加了 HTML 的 style 属性的重要性、以及 clip 属性的一个新的计算方法），同时增加了一些呼声最高，并且在很多已经投入了使用的功能。但是 CSS 2.1 大部分还是一个 CSS 使用的“快照”：它包含了所有大家都推荐的功能。

CSS 2.1 从 CSS 2 中继承而来，并且会替换掉 CSS 2 。CSS 2 当中的一部分被保留了下来，一部分被修改了，还有一部分被删除了。被删除的某些功能，可能会在 CSS 3 的规范中加上。将来所说的规范，将默认指 CSS 2.1 （除非需要一些在 CSS 2.1 中被删除掉的 CSS 2 的功能，或者在 CSS 3 模块中参考这些功能）
